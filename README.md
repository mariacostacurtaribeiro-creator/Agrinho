# 🚀 AgroTec: conexão entre o Agronegócio e a Tecnologia

---

## 🎯 Objetivos

O projeto foi desenvolvido com os seguintes propósitos:

* Conscientizar as pessoas acerca da integração da tecnologia para com o meio ambiente;
* Incentivar práticas mais responsáveis e conscientes;
* Exemplificar mudanças benéficas a partir da inserção da tecnologia no meio ambiente.

---

## 🛠️ Recursos Técnicos

Tecnologias, linguagens e ferramentas utilizadas no desenvolvimento:

* HTML;
* CSS;
* JavaScript;
* GitHub
